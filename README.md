# MHWs_AutoBackup
이는 몬스터헌터 와일즈 실행 시 데이터 손상으로 인한 날림 현상을 방지하기 위한 자동 백업 bat파일입니다.



# 적용 방법
* 본인 스팀 몬스터헌터 와일즈 로컬 경로 파일에 위 .bat 추가.

  
* set UserID=  에 스팀 ID 입력
* set BackupFolder=   에 백업을 저장할 폴더 경로 추가 후
